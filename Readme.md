# UltronX – One Tap Study Group 

## About Our Project
Studying alone can sometimes feel difficult and unmotivating. Many students want to study in groups but don't know how to find the right people or manage time together.
One Tap Study Group is our attempt to solve this problem by making it easy for students to connect and study together.

Our idea is simple: with just one tap, students can create or join study groups, share resources, and stay productive together.

## Problem We Are Solving
Students often face problems like:
- Not finding serious study partners
- Difficulty in coordinating group study
- Lack of a common platform for collaboration
- Poor communication in study groups

## Our Solution
We built a simple platform where students can:
- Create study groups easily
- Join groups based on subjects
- Plan study sessions
- Collaborate in a simple and organized way

## Key Features 
- Quick study group creation
- Easy group joining
- Subject based matching
- Study scheduling
- Simple and user friendly interface

Install dependencies:

npm install

Run the project:

npm start
